# 快速启动指南

## 1. 安装依赖

```bash
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

访问: http://localhost:5173

## 4. 功能测试流程

### 注册流程
1. 访问注册页面
2. 填写用户信息（需要邀请码）
3. 提交注册
4. 等待客服审核

### 交易流程
1. 登录系统
2. 查看实时行情
3. 点击"买入"或"卖出"
4. 填写价格和数量
5. 确认下单
6. 查看订单状态

### 充值流程
1. 进入"资金"页面
2. 点击"充值"
3. 填写充值金额
4. 选择银行卡
5. 提交申请
6. 等待审核

### 提现流程
1. 进入"资金"页面
2. 点击"提现"
3. 填写提现金额
4. 选择银行卡
5. 提交申请
6. 等待审核

## 5. 注意事项

### 后端服务
确保后端服务已启动: http://localhost:8080

### WebSocket连接
系统会自动连接行情WebSocket，查看浏览器控制台确认连接状态。

### 数据库
首次运行需要初始化数据库，参考后端README。

## 6. 常见问题

### 无法连接后端
检查后端服务是否启动，API地址是否正确。

### WebSocket连接失败
检查后端WebSocket服务，确认端口和协议正确。

### 登录失败
确认账号密码正确，查看后端日志。

### 页面空白
打开浏览器控制台查看错误信息。
